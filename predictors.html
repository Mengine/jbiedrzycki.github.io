Choose Your Predictors

The purpose of this application is to fit a linear model for mtcars data set where mpg is an outcome and predictors are selected by the user.

The program presents accuracy of the model (r squared), some diagnostic plots for linear regression and a table of coefficiencies. By default all of the variables are used as predictotrs.

I/O

Input

  • A list of predictors

Output

  • Model accuracy
  • Diagnostic plots
  • Coefficients

How it works

The heart of the application

model_predictors<-paste(selected_predictors,collapse="+")
model_formula<-paste("mpg",model_predictors,sep="~")

fit<-lm(model_formula,mtcars)
fit_coef<-summary(fit)$coefficients
